Loan Approval Prediction & Maximum Loan Amount Estimation
Project Overview

This machine learning project was developed as part of the Machine Learning & Data Mining coursework. The objective was to build predictive models that:

Predict whether a loan application will be approved.
Estimate the maximum loan amount that can be offered to approved clients.

The project follows the CRISP-DM methodology and includes data preparation, classification modelling, hyperparameter tuning, ensemble learning, regression modelling, and performance evaluation.
